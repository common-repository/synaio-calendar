Synaio Calendar est un plugin destiné aux utilisateurs de la fonction "Événement" de l'outil Synaio.

== Installation ==
Comment installer Synaio Calendar ?

1. Appuyer sur le bouton télécharger.
2. Extraire l'archive .zip dans votre dossier `wp-content/plugins`. Ou bien allez dans l'onglet Extensions > Ajouter de votre WordPress, puis rechercher "Syanio Calendar"
3. Dans l'onglet Extensions > Extensions installées et Activez votre plugin
